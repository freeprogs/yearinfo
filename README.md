
# yearinfo

Counts for the given date yin and yang duality, classical element and
an animal in the Chinese calendar.

---

### Requirements

This application has tested in Linux Fedora 26 with Qt5.


### Building

Build the docs and read the README file in _build/docs_.

To build run:

```sh
$ ./configure
$ make
```

### Testing

To build tests run:

```sh
$ ./configure
$ make tests
```

### Installation

After installation search the program in the KDE menu or you can run it by name _yearinfo_.

To install run:

```sh
$ sudo make install
```

To uninstall run:

```sh
$ sudo make uninstall
```
